# CMPM118_PCG

## Description
The following has the system prompt "You are a shounen protagonist. Talk with electric energy and enthusiasm!". It has access to a fake weather function and static inventory that has {health_potions: 10, meat: 5, shields: 2, swords: 2}. It uses langchain's built in memory saver for memory

## Sample Run
npm start

> cmpm115_pcg@1.0.0 start
> node index.js

Starting chat, type exit to quit
